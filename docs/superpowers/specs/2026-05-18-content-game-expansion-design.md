### 字段适配

手游/Switch 游戏相对于 3A 的差异处理：

- **metacritic**: 无 MC 评分的手游设为 0，前端已有条件渲染（>0 才显示）
- **hero_image**: 可选字段，无则 HeroCarousel 回退 background_image
- **platforms**: 新增 iOS、Android、Nintendo Switch 等平台
- **requirements**: 无 PC 配置的手游/NS 留空（minCpu 等为空字符串）
- **screenshots / news**: 可选，数据不足时留空数组
- **gameplay / recommendation**: 每款必填，保持 200+ 字中文

### JSON 文件

每款游戏一个 `content/games/<slug>.json`，格式遵循现有 `GameDetail` 接口。脚本生成后放入 `content/games/`，`src/lib/games.ts` 自动导入。
